Clone & Noise Detection

Advanced image tampering detection using multiple algorithms including copy-move detection, block matching, noise analysis, and statistical anomaly detection.

Detection Methods (for comprehensive analysis)
Copy-Move Detection

Detects identical or similar regions that have been copied and pasted within the same image using SIFT/ORB feature matching.

Block Matching

Analyzes image blocks for similarities using normalized cross-correlation to identify cloned regions.

Noise Analysis

Examines noise patterns and consistency across different regions to detect tampering artifacts.

Statistical Analysis

Analyzes statistical properties (mean, std, skewness, kurtosis, entropy) for anomaly detection.
